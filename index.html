Termo

Termo é um jogo de palavras no qual o jogador deve acertar uma palavra dentro de um número de tentativas e com base em dicas no comportamento das letras da palavra digitada. Quando as letras aparecem na cor verde, além de existirem na palavra alvo, elas estão na posição correta. As amarelas sinalizam que as letras em questão compõem a palavra, porém, estão na posição errada. Já as que aparecem brancas, não fazem parte da palavra.

Software License

Este projeto consiste em um sistema de licenciamento de software que utiliza um contrato inteligente (smart contract) escrito em Solidity, uma linguagem de programação para contratos inteligentes na blockchain Ethereum. O objetivo principal do contrato inteligente é facilitar a venda e distribuição de licenças de software de maneira descentralizada e segura.

Click TCP

Escrito em Python, utilizando o framework kivy, o ClickTCP é um jogo simples desenvolvido com o objetivo de colocar em prática alguns conceitos de redes de computadores ao implementar sockets para a comunicação usando protocolo TCP/IP. Nele, a janela é dividida em duas porções, cada uma com uma cor diferente e representando um jogador. Conforme um jogador acumula uma vantagem no número cliques em relação a seu adversário a sua porção da tela aumenta até que a tela toda seja ocupada e ele vença a partida.
